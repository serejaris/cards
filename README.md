# ЧГК Cards — Личная база фактов и генератор flash cards

Прототип системы для сбора, хранения и повторения фактов перед играми в «Что? Где? Когда?».

## Идея

Помогает быстро запоминать разрозненные факты, культурные связи, имена и даты. Система позволяет строить личную базу знаний и генерировать на её основе классические flash cards (вопрос → ответ → пояснение).

## Контекст урока

Проект появился на уроке **AI Mindset {X26}** 20 мая 2026 года. На занятии разбирался путь от сырой идеи до первого рабочего прототипа:

1. Сформулировать идею голосом или текстом в чат-боте.
2. Получить PRD: аудитория, MVP, сущности, сценарии и экраны.
3. Скачать PRD как `prd.md` и положить в папку проекта.
4. Попросить агента сделать быстрый HTML-прототип.
5. Перенести проект в GitHub, опубликовать и дальше улучшать через итерации.

Идея для этого проекта: личная база фактов и генератор flash cards для подготовки к ЧГК. Первый полезный результат — файл с описанием идеи и прототип, который уже можно открыть, показать и критиковать.

```text

Вопросы задавай строго по одному.

В конце верни мне PRD этой идеи и промпты, которые помогут мне реализовать MVP вместе с моим ИИ-агентом в Кодексе.
```

```text

```

## Дизайн

Интерфейс **Classic Study** — чистый академический стиль: белый фон, индиго для акцентов, скруглённые карточки.

## Как запустить

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 8765
```

Затем откройте http://localhost:8765/

## Технологии

- React
- Vite
- Tailwind CSS (CDN)
- localStorage

## Основные возможности (MVP)

- Создание тем и добавление фактов
- Генерация карточек из фактов
- Режим тренировки (знала / не знала / повторить)
- Работа с ошибками после игр
- Статистика по слабым темам

## Структура

- `index.html` — Vite entrypoint
- `src/App.jsx` — верхний state и навигация
- `src/screens/` — экраны приложения
- `src/components/` — общие компоненты
- `src/data/seed.js` — стартовые данные
- `src/lib/storage.js` — localStorage
- `prd.md` — product requirements document
